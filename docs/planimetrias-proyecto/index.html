Entrega 2 - Materiales Ecológicos

Impresión de moldes en 3D

Para lograr el molde específico que necesitamos, primero debemos imprimirlo, así nos aseguramos de que cada vez que lo utilicemos lograremos el mismo resultado.

El material ideal para imprimir estos moldes es PLA+, ya que este nos asegura una alta resistencia y fácil usabilidad.

Existen muchos ejemplos en internet de cómo funciona esta técnico, a continuación dejaremos una imágen de un molde que encontramos para luego hacer maceteros de concreto.

Planimetrías

En base a la corrección de la clase pasada, agregamos una inclinación de 10 grados a la superficie que entra en contacto con el molde para hacer más fácil el desmoldado.
